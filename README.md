# Clicky
A custom mouse case with precise cutouts for buttons, the scroll wheel, and screws, featuring a custom shape made specifically for the Bambu Lab Wireless Mouse Components Kit.This design is made because ,I simply want a custom mouse for myself with my own design. 

## How to Use :
- Insert the Component Kit inside the Custom 3D Case.
- Insert and tighten the bottom screws to secure it.
- Place 1*AA Battery inside and use it. :)
